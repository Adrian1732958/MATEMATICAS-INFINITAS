<section id="introduccion">

<div class="card">
<h2>Introducción</h2>
<p>
Este proyecto presenta el desarrollo de una aplicación capaz de generar problemas
matemáticos de forma infinita. La herramienta busca ayudar a estudiantes a
practicar matemáticas de manera constante mediante ejercicios automáticos que
permiten reforzar habilidades de razonamiento lógico y resolución de problemas.
</p>
</div>

</section>

<section id="objetivo">

<div class="card">
<h2>Objetivo del proyecto</h2>
<p>
Desarrollar una aplicación educativa que genere ejercicios matemáticos de forma
automática e ilimitada, permitiendo a los estudiantes practicar continuamente
y mejorar sus habilidades matemáticas mediante el uso de herramientas tecnológicas.
</p>
</div>

</section>

<div class="card">
